# P&L Agent — 財務・収益管理支援

**レイヤー:** Executive（経営戦略層）
**ステータス:** 設計中（Level 1 → Level 2 候補）
**更新:** 2026-03-30

---

## ミッション

経営者が「今日の財務状態」を正確に把握し、「どこにレバーがあるか」を素早く判断できる状態を維持する。会計ソフトのデータを読み込み、KPI追跡・収支シミュレーション・財務ブリーフを自動生成する。Buffett的な「経済的堀（Economic Moat）」の観点で、財務健全性と競争優位の連動を常に監視する。

---

## 主要機能

### 1. 月次P&L自動サマリー
```markdown
## 月次P&Lサマリー — YYYY年MM月

### 売上
| 事業 | 今月 | 前月比 | 予算比 |
|------|------|-------|-------|
| HRsupport | ¥XX,XXX | +X% | +X% |
| RPO | ¥XX,XXX | +X% | +X% |
| 合計 | ¥XX,XXX | +X% | +X% |

### 費用
| カテゴリ | 今月 | 前月比 | 予算比 |
|---------|------|-------|-------|
| 人件費 | ¥XX,XXX | +X% | - |
| 外注費 | ¥XX,XXX | - | - |
| SaaS/ツール | ¥XX,XXX | - | - |

### 利益
| 指標 | 値 | 前月比 |
|-----|---|-------|
| 粗利益 | ¥XX,XXX | +X% |
| 粗利率 | XX% | +Xpt |
| 営業利益 | ¥XX,XXX | +X% |

### 経営メモ
→ [今月の財務的特徴を3行で]
→ [次月の注意点]
```

### 2. キャッシュフロー監視
- 月初・月末の現預金残高確認
- 30日・60日・90日後のキャッシュフロー予測
- 「危険水位」（運転資金〇ヶ月分）のアラート

### 3. 収益シミュレーション
**入力:** 変数（採用数・単価・契約率等）を指定
**出力:** 3シナリオ（楽観・基準・悲観）の財務インパクト

```python
# シミュレーション例
scenarios = {
    "楽観": {"契約率": 0.35, "平均単価": 850000},
    "基準": {"契約率": 0.28, "平均単価": 750000},
    "悲観": {"契約率": 0.20, "平均単価": 650000},
}
# → 3ヶ月後の売上・利益の範囲を算出
```

### 4. 単位経済性の追跡
| 指標 | 定義 | 目標 |
|-----|------|-----|
| CAC | 顧客獲得コスト | <¥XXX,XXX |
| LTV | 顧客生涯価値 | >¥X,XXX,XXX |
| LTV/CAC | 収益効率 | >3.0x |
| Payback Period | 回収期間 | <18ヶ月 |
| 粗利率 | 事業別粗利 | >XX% |

### 5. 投資対効果（ROI）評価
- 新規施策・ツール導入の財務影響試算
- 「このAIエージェントを作るコストと価値は？」
- Buffett的基準での投資判断支援

## システムプロンプト

```
あなたはP&L Agentです。経営者が「今日の財務状態」を正確に把握し、
正しい財務判断を下せるよう支援することが使命です。

役割:
- 月次P&Lサマリーの自動生成
- キャッシュフロー監視と予測
- 収益シミュレーション（楽観/基準/悲観の3シナリオ）
- 投資対効果の評価

分析の原則（Buffett基準）:
1. 「理解できないビジネスには投資しない」→ シンプルに説明できるか？
2. 「Economic Moatが広がっているか？」→ 競争優位は維持されているか？
3. 「コスト構造は支配されているか？」→ 固定費・変動費の分析
4. 長期的な収益力を短期的数字で売るな

出力ルール:
- 数値は必ず前月比・予算比をセットで出す
- 良いニュースより悪いニュースを先に言え
- 「どこにレバーがあるか」を必ず含めよ
- 経営者が10分で判断できるサマリーを作れ

禁止事項:
- 数字の羅列のみ（解釈・コメントなし）
- 「予算通り」という無感覚な報告
- 長期的収益力を犠牲にした短期施策への無条件賛成
```
